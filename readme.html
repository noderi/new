Installation

  1. Download LicenseBot
  2. Unzip it to your desired location/folder
  3. Fill in config/config.json to your likings with a text editor.
  4. Install the dependencies. Run npm install --only=production
  5. Done! To start the bot, run npm start

Upgrading

  1. Download the new version
  2. Back up your database and config/config.json
  3. Extract the files into your bot directory
  4. Fill in your new config/config.json to your likings. Make sure dbinit is set to true!
  5. Delete folder node_modules and file package-lock.json
  6. Run npm install to install/update the dependencies
  7. Execute the file LicenseBot.sql in your database
  8. Set dbinit in config/config.json to true (ONLY IF YOU ARE UPGRADING)
  9. Done. Run npm start to start the bot!
